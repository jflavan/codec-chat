## Color Scheme

Based on the existing CODEC CRT theme:

- **Background:** `--bg-primary` (dark gray)
- **Backdrop:** `rgba(0, 0, 0, 0.85)` (semi-transparent black)
- **Text:** `--text-normal` (light gray)
- **Headers:** `--text-header` (white)
- **Accent:** `--accent` (bright cyan/green)
- **Borders:** `--border` (subtle gray)
- **Input Background:** `--bg-secondary` (slightly lighter gray)
- **Hover State:** `--bg-message-hover` (subtle highlight)

## Animation & Transitions

- **Modal Open:** Fade in backdrop (150ms ease)
- **Modal Close:** Fade out backdrop (150ms ease)
- **Button Hover:** Color transition (150ms ease)
- **Input Focus:** Border color transition (150ms ease)
- **Edit Mode Toggle:** Smooth height transition

## Accessibility Indicators

- **Focus Outline:** Visible on all interactive elements
- **ARIA Labels:** 
  - Modal: `aria-label="Server Settings"`
  - Close button: `aria-label="Close settings"`
  - Input fields: Associated `<label>` elements
- **Keyboard Shortcuts:**
  - ESC: Close modal
  - Enter: Submit forms
  - Tab: Navigate between elements

## Real-time Update Indicators

When a change is made:
1. **Loading State:** Button shows "Saving..." text, disabled state
2. **Success State:** Form exits edit mode, new value displayed
3. **Error State:** Error message displayed (via transient notification)

## Future Visual Enhancements

Potential UI improvements:
1. **Success Toast:** Brief "Saved!" notification
2. **Loading Spinner:** Visual feedback during API calls
3. **Undo Button:** Quick revert for accidental changes
4. **Drag Handles:** For future channel reordering
5. **Delete Icons:** For future channel deletion
6. **Preview Mode:** Live preview of server name changes
